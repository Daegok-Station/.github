# DevOps CI/CD pipeline 구축

---

## 프로젝트 소개
* 본 프로젝트는 **Spring PetClinic** 애플리케이션을 배포 대상으로 선정하여, 개발부터 운영까지의 전 과정을 자동화하는 DevSecOps CI/CD pipeline을 구축합니다.
* **Jenkins**를 엔진으로 사용하고, **SonarQube**와 **Trivy**를 보안게이트로 통합하여, AWS ECS 기반의 **Blue/Green 배포** 방식을 채택했습니다.
* **Prometheus/Grafana를 통해 운영 단계의 실시간 모니터링 환경 구축까지를 목표로 하고 있습니다.

---

## 프로젝트 목표
* CI/CD pipeline의 핵심 기능의 이해 / 완성
* (Cloud 기반)DevOps 문화 및 기술 습득
* 운영 단계에서 발생할 보안 취약점 사전 차단 환경 구축
